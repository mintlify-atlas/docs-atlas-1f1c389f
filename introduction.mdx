Query your message history, search across conversations, find attachments, and send messages — all through natural language, all running locally on your Mac.

## How it works

The skill reads directly from macOS's local SQLite databases — no cloud sync, no third-party APIs. Your messages never leave your Mac (beyond what Claude Code normally processes).

```
/imessage-mail who texted me
/imessage-mail texts with Sarah
/imessage-mail search "project deadline"
/imessage-mail email
/imessage-mail send a text to Mike saying I'm running late
/imessage-mail find attachments from David
```

<Note>
This skill requires **Full Disk Access** granted to your terminal application in macOS System Settings. See [Requirements](/requirements) for setup details.
</Note>

## Privacy

All database queries run entirely on your Mac. No message content, contacts, or email data is sent to any external service beyond what Claude Code normally processes. The send functionality uses AppleScript to control Messages.app and Mail.app locally.
